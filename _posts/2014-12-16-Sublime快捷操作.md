##Sublime快键操作
###1. Go to Anything
快捷键 Command + P

* 在项目文件中快速切换，sublime支持模糊匹配，文件切换很方便
* 在一个文件中快速寻找到目标元素：@ + 想要寻找的元素，方向键进行选择

###2. 多行游标
操作方式1:

选中要选择的单词

Command + D

如果选中了不想要的部分，Command + K 跳过，然后继续 Command + D 

操作方式2:鼠标右键拖拽


###3. 命令模式
启动命令模式: shift + Command + P
